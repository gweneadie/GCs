OK, now I will try calculating the mass profile. Physically the mass contained within some distance \(r\) of the cluster can be found by integrating the mass density \(\rho (r)\) and assuming spherical symmetry \[ M(r<R) = 4\pi\int \rho (r) r^2 dr \]

We dont have an analytic expression for \(\rho (r)\), unfortunately, so we will have to numerically integrate. The limepy model returns a set of \(N\) pairs pf \(r\) and \(\rho (r)\) values. The integrand is \(\rho (r) r^2\), so lets plot this function to see how it looks:

testpars <- target$chain[2, ]
testpars
##            g        Phi_0            M          r_h 
## 1.438646e+00 5.541668e+00 1.261819e+05 2.997227e+00
# calculate model for these parameter values
exmodel <- limepy$limepy(g = testpars[1], phi0 = testpars[2], M = testpars[3], rh = testpars[4])
# plot the mass density profile
plot(exmodel$r, exmodel$rho*exmodel$r^2, xlab="r (pc)", ylab=expression(rho(r)))

This doesnt look too terrible, but we also dont want to integrate with so few points. It is a smooth function, so we will fit a spline to this function and then numerically integrate that spline. As an example, here is how well a spline fit to the points above can represent this function of \(\rho (r) r^2\):

testspline <- splinefun(x = exmodel$r, y = 4*pi*exmodel$rho*exmodel$r^2, method = "natural")
par(mar=c(5,6,2,2))
plot(exmodel$r, 4*pi*exmodel$rho*exmodel$r^2, xlab="r (pc)", ylab=expression(rho(r)*r^2), cex.lab=1.5, cex.axis=1.5)
grid()
curve(testspline, n=1e4, col="blue", add=TRUE)

To integrate the spline and therefor obtain the mass within a variety of radii \(r\), I use the function integrate(). For each \(r_i\) returned by limepy (except \(i=1\), which is 0), we integrate the spline function from \(r=0\) to \(r_i\). For each \(i\), the section is subdivided into 100.

# make a new column in the data frame
exmodel$mass <- 0
for(i in 2:length(exmodel$r)){
  exmodel$mass[i] <- integrate(f = testspline, lower = exmodel$r[1], upper = exmodel$r[i], subdivisions = 100)$value
}

Now we can plot the mass profile:

par(mar=c(5,6,2,2))
plot(exmodel$r, exmodel$mass, xlab="r (pc)", ylab = expression(M(r<R)~(M['\u0298'])), cex.lab=1.5, cex.lab=1.5, type="l")

Next, we will do this for every set of parameters in the Markov chain! (It took maybe 30 minutes for this to run, so not run here). We will also calculate the median of the total mass parameter from the posterior samples and include this as a horizontal line on the figure.

# calculate a mass profile for every set of parameter values from the posterior distribution samples
allmassprofiles <- apply(X = target$chain, MARGIN = 1, FUN = massprofile)

# save because that took a while!
saveRDS(allmassprofiles, file = paste0("../results/massprofiles_", Sys.Date()))

# max M for plotting
bigmassdataframe <- bind_rows(allmassprofiles, .id = "column_label")
maxM <- max(bigmassdataframe$mass)

# plot these
png(filename = paste0("../results/Figures/mass_profile_", Sys.Date(), ".png"), width = 900)

par(mar=c(5,6,2,2))
plot(allmassprofiles[[1]]$r, allmassprofiles[[1]]$mass, type="l", xlab="r (pc)", ylab=expression(M(r<R)~(M['\u0298'])), cex.lab=2.5, cex.axis=2, xlim = c(0,maxr), ylim=c(0,maxM))
grid()
lapply(X = allmassprofiles, FUN = function(X) lines(X$r, X$mass, col=rgb(0,0,0,0.01)) )

abline(h=median(target$chain[,3]), col="blue", lty=2)

dev.off()
